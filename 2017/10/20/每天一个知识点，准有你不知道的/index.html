关键词： Object.assign() 方法

  • 这个方法是ES6中新增的,用于将所有可枚举属性的值从一个或多个源对象复制到目标对象,它将返回目标对象。
  • 语法： Object.assign(target,…sources)
  • 参数：target-目标对象;sources-源对象
  • 返回值：目标对象
  • 描述：
    • 如果目标对象中的属性具有相同的键，则属性将被源中的属性覆盖。后来的源的属性将类似地覆盖早先的属性。
    • Object.assign 方法只会拷贝源对象自身的并且可枚举的属性到目标对象。
    • Object.assign 会跳过那些值为 null 或 undefined 的源对象。

  • 栗子

    • 复制对象

    • 针对深拷贝，需要使用其他方法，因为 Object.assign()拷贝的是属性值。假如源对象的属性值是一个指向对象的引用，它也只拷贝那个引用值。

    • 合并对象

    • 继承属性和不可枚举属性是不能拷贝的

    • 原始类型会被包装为对象

关键词：Notification API

  • Notification的意思是’通知’’提示’，顾名思义，Notification API是html5中新增的桌面通知的API，由于最近学校的课程设计需要做一款网页聊天平台，所以了解到了这个API。
  • 在web页面中我们经常可以收到消息提醒的通知，在之前比较早的bbs中，有新消息通知一般都是在网页标签页的title中进行闪烁提醒的，原理很简单，就是通过定时器修改document.title
  • Notification API的通知是和浏览器脱离的，而且是置顶的，就想某360的弹窗一样（…没有黑的意思）,所以哪怕你最小化了浏览器，消息都会置顶显示。
  • 这么强大的功能肯定是需要用户授权的：Notification.permission表示当前通知显示的用户授权状态，有三个值：
    • default ：不知道用户的选择，默认。
    • granted ：用户允许。
    • denied ：用户拒绝。

关键词：’data-‘ 属性

  • html5中新增的用户自定义属性，有两种设置的方式
    • 最常用的就是直接写在标签中: data-xxx=”xxx”
    • 也可以用过元素的dataset属性在js中设置：obj.dataset.xxx=”xxx”
  • 获取也有两种方式：
    • obj.dataset.xxx
    • obj.dataset[xxx]
  • 需要注意的是如果自定义的属性中包含’-‘,那么在获取是应该使用驼峰法
    • data-my-arr=”xxx”
    • 获取: obj.dataset.myArr或obj.dataset[‘myArr’]
