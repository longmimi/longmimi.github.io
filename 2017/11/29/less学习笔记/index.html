less介绍

less是一门CSS预处理语言，它扩充了CSS语言，增加了诸如 变量，继承，运算 ，混合（mixins），函数，作用域，嵌套规则等功能，让CSS更加容易编写和维护。

注释

  • /* */ 会被编译保留在css中
  • // 不会被编译css中不会保留

变量

  • 和js中的变量一样，只是less的变量定义不是使用 var 而是使用 @

混合

  • 我的理解是，混合就是把别的选择器的样式拿到另一个选择器中使用

  • 编译之后生成

  • 但是这样全部拿过来用难免有些死板，于是有带参混合

  • 编译后

  • 注：带参混合也可以给默认值，那么使用的时候，如果参数为空，那编译出来就是给的默认值，如果参数不空，那编译出来就是自己给的值

匹配模式

  • 我的理解是写多个同名的class,然后传不同的参数，按照参数来使用less

  • 可以看到上面有大部分重复代码，所以可以将重复代码抽离出来再写一个class,这就短一些了，注意那个@_

运算

  • 任何变量，数字，颜色都可以参数运算 这没啥说的，颜色都可以加减。。。
